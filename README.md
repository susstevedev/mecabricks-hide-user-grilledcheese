# mecabricks-hide-user-grilledcheese
Removes user Grilledcheese from Mecabricks forum posts and comments with CSS.
# How to install
Install [TamperMonkey](https://www.tampermonkey.net) or [Violentmonkey](https://violentmonkey.github.io).
Then click [here](https://github.com/susstevedev/mecabricks-hide-user-grilledcheese/raw/main/index.user.js). You should see an installation prompt. Click install.
